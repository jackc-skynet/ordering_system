# Change Set Summary: v1.4.0 - Shop Location & Distance Page

**Date**: 2026-03-19
**Version**: v1.4.0

## Affected Files
- `src/ShopLocation.jsx`: [NEW] Interactive map component using Leaflet.js.
- `src/App.jsx`: Integrated the new location view and added a "Find Us" navigation button.
- `package.json`: Added `leaflet` and `react-leaflet` dependencies.

## Key Changes
- **Interactive Map**: Added a new independent page featuring a high-quality Leaflet map. The map is centered on the shop's location at Taipei 101 and uses a custom dark theme to match the system's aesthetic.
- **Real-time Distance Calculation**: Implemented the Haversine formula to calculate the "as-the-crow-flies" distance between the user's current geolocation and the shop.
- **Route Planning**: Integrated an easy-to-use "Get Directions" button that opens Google Maps with the destination pre-filled.
- **Premium UI**: Designed a glassmorphism info card that provides shop details and distance information with a modern, elegant look.
- **Navigation Integration**: Added a "門市定位" (Find Us) button in the main menu header for seamless access.
