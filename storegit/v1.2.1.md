# Change Set Summary: v1.2.1 - Persistent Order Tracking Entry

**Date**: 2026-03-19
**Version**: v1.2.1

## Affected Files
- `src/App.jsx`: Added `FloatingStatusButton` and implemented state logic for toggling tracking view visibility.

## Key Changes
- **UX Accessibility**: Introduced a Floating Action Button (FAB) that appears when an order is active but the user is browsing the menu.
- **Seamless Toggling**: Allows users to jump back and forth between the menu and the real-time tracking screen without losing progress.
- **Dynamic Feedback**: The FAB changes color and pulses when the order is "Ready for Pickup", providing ambient notification to the customer.
