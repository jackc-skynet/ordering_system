# Change Set Summary: v1.2.0 - Real-time Order Tracking

**Date**: 2026-03-19
**Version**: v1.2.0

## Affected Files
- `src/App.jsx`: Implemented `OrderTrackingView` with Supabase Realtime subscription.
- `src/Admin.jsx`: Updated status workflow and UI buttons to support `preparing` and `ready` states.

## Key Changes
- **Customer Experience**: The static success screen is now a live "Wait Screen" that tracks order progress.
- **Kitchen Workflow**: Added a `ready` state to explicitly notify customers when their food is ready for pickup.
- **Real-time Sync**: Used Supabase `.channel().on('postgres_changes', ...)` to ensure sub-second UI updates across devices.
