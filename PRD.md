# 產品需求規格書 (PRD)：點菜系統 (Ordering System)

## 1. 產品概述
這是一款現代化的點菜應用程式，旨在提供流暢、美觀且直覺的用餐點餐體驗。使用者可以瀏覽菜單、選擇份量/選項，並將餐點加入購物車後送出訂單。

## 2. 核心功能
### 2.1 菜單瀏覽 (Menu Browsing)
- **分類導航**：上方提供分類標籤（如：主食、點心、飲品、甜點），點擊後快速過濾。
- **餐點卡片**：顯示高品質餐點圖片、售價、以及簡短描述。
- **動態效果**：滑過或點擊時具備微互動動畫。

### 2.2 購物車管理 (Cart Management)
- **即時更新**：點擊「加入」後，底部購物車欄位即時顯示數量與金額。
- **品項調整**：在購物車中可以增減數量或移除品項。
- **金額計算**：自動計算小計與總計。

### 2.3 訂單提交 (Order Placement)
- **模擬下單**：點擊下單後顯示動態處理中的動畫，最後彈出「訂單成功」頁面。

## 3. 設計美學 (Design Aesthetics)
- **配色**：使用深色背景搭配鮮豔的主題色（如：暖橙色或深翠綠），營造進階感。
- **效果**：運用毛玻璃 (Glassmorphism) 效果於導航欄與購物車。
- **字體**：選用現代無襯線字體（如：Inter 或 Outfit）。

## 4. 技術規格
- **前端**：React + Vite
- **樣式**：純 CSS (Modern Vanilla CSS)
- **響應式**：優先適應手機端，同時兼顧桌機版配置。
